# midi2otomad

単体で完結する GUI を備えた、MIDI 音MAD作成デスクトップアプリ。
`.mid` を読み込み、トラックごとに WAV/MP3 などの音声素材を割り当て、ノートの音高に合わせて
ピッチ（再生速度）を計算して発音し、1 本の高音質 WAV / MP3 として一瞬で書き出します。

## 主な機能

- **MIDI 取り込み** — トラック・ノート・テンポ・CC7/CC11・ピッチベンドを解析。GM ch10 のドラムトラックは音程固定を自動 ON。
- **素材ごとの音作り** — 基準ピッチ（自動検出つき）・DAHDSR エンベロープ・biquad フィルター（EG/LFO 連動）・ダイナミックピッチ（グライド/ビブラート）。
- **ベロシティマッピング** — 効き・カーブ・フィルターカットオフへの追従を素材ごとに調整。
- **タイムストレッチ** — 音程を保ったまま、ループ無しでワンショットをノート長へ伸ばす（WSOLA）。
- **ワンショット位相補正** — 再生開始を立ち上がりへ揃え、DC オフセットを除去。
- **ピッチベンド / 音程固定** — トラック全体のベンド（レンジ可変）と、ドラム/ワンショットキット向けの音程固定。
- **マスター処理** — Freeverb リバーブとソフトクリップ・リミッターを通し、WAV / MP3 へ書き出し。
