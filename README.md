# midi2otomad

単体で完結する GUI を備えた、MIDI 音MAD作成デスクトップアプリ。
`.mid` を読み込み、トラックごとに WAV/MP3 などの音声素材を割り当て、ノートの音高に合わせて
ピッチ（再生速度）を計算して発音し、1 本の高音質 WAV / MP3 として一瞬で書き出します。

## 主な機能

### 取り込み
- `.mid` のドラッグ＆ドロップ読込 — トラック / ノート / テンポ変化 / CC7・CC11 / **サスティンペダル (CC64)** を解析
- ドラムトラックの自動判定（GM ch10・バンク127）と、音階/ドラム混在トラックの自動分離
- WAV / MP3 / OGG / FLAC / AAC などの音声素材を一括読込（波形サムネイル付き）

### 素材エディタ
- **波形ドラッグでトリム / ループ範囲を直接指定**、スライダーでの微調整も可能
- 基準ピッチの自動検出（YIN）とスケール試聴チップ（±半音で即試聴）
- テープ方式（音程＝速度）と**タイムストレッチ方式**（グラニュラー、音程と速度を分離）
- リバース再生・ワンショット・DAHDSR エンベロープ（カーブ調整付き）
- 音色フィルター（8 タイプ、EG スイープ + LFO ワブル）・グライド / ビブラート
- 複数素材のレイヤー再生（1 ノートで重ねて発音）

### トラック / ミックス
- ノート番号ごとの素材差し替え（ドラムキット構築）と**音高ごとの試聴ボタン**
- 演奏表現プリセット — 音程（ピッチ）と抑揚（強弱）を独立して反映 / 固定
- 最大同時発音数・優先度・停止方法（同音程 / 同素材 / トラック）のボイス管理
- トラック音量 / パン / リバーブ送り、マスターリバーブ（Freeverb）、マスターリミッター

### スタジオ
- **プロジェクトの保存 / 読み込み (.m2oproj)** — 素材の参照パスも保存し、開くだけで復元
- **Undo / Redo**（Ctrl+Z / Ctrl+Shift+Z / Ctrl+Y、スライダー操作は自動でまとめて 1 手順に）
- **ループ再生**トグル・Space で再生 / 停止・タイムラインクリックでシーク
- ピアノロールに**小節グリッド**（テンポマップ追従、4/4 想定）
- クリップ警告付きレベルメーター、軽量プレビュー用の高速モード
- WAV (16/24/32bit float) / MP3 (192–320kbps) 書き出し

## ショートカット

| キー | 動作 |
| --- | --- |
| `Space` | 再生 / 一時停止 |
| `Ctrl+Z` / `Ctrl+Shift+Z` / `Ctrl+Y` | 元に戻す / やり直す |
| `Ctrl+S` | プロジェクトを保存 |

## 開発

```bash
# 依存: Rust (stable) + wasm32-unknown-unknown ターゲット + trunk + Tauri 2 の系統依存
rustup target add wasm32-unknown-unknown
cargo install --locked trunk

# テスト
cargo test -p midi2otomad-core
cargo test -p midi2otomad-ui
cargo test -p midi2otomad

# 開発起動（フロントエンドは trunk が自動でビルド）
cargo tauri dev

# リリースビルド
cargo tauri build
```

構成: `core/`（純 Rust の合成エンジン — スキーマ / DSP / MIDI / メディア I/O）、
`ui/`（Leptos + WASM フロントエンド）、`src-tauri/`（Tauri 2 シェルと cpal 再生）。
